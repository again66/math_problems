数学学习工具

针对低年级口算训练打造的微信小程序，支持自动出题、错题本记录、打印练习单等功能，帮助孩子在家高效巩固数学能力。

微信扫码体验小程序

核心亮点

针对家庭学习场景设计

结合真实使用需求打造的轻量工具，支持不同年级的口算训练，并且提供进阶的错题管理与打印能力。

智能出题

一键生成加减乘除与混合运算题，可配置题量、进位/借位、数值区间等参数。

错题本

内置错题回顾组件，自动记录做错的题目，方便针对性复习。

打印支持

网页端可导出打印版题目，适合离线练习或课堂发放。

动画激励

提供趣味动效与 emoji 奖励，提升孩子做题积极性。

完全离线

不采集、不存储个人数据，每次进入后按需配置题库即可。

开源共享

完整源代码开放，可自行二次开发或搭建私有版本。

使用流程

3 步快速开始

  1. 1

    微信扫码

    在微信中扫描页面上的体验码或搜索“小学数学学习工具”进入小程序。

  2. 2

    配置题目

    选择需要的题型、题量和范围，系统会自动生成符合条件的题目。

  3. 3

    练习与复盘

    在线作答或打印练习单，错题自动进入错题本，随时复习。

技术栈

TypeScript + SCSS + WeChat Mini Program

代码结构遵循微信小程序最佳实践，支持 TypeScript 类型提示与 Sass 预处理，易于维护与扩展。

TypeScript

严格的类型定义，降低逻辑错误，方便 IDE 智能提示。

SCSS

模块化样式组织，支持变量与混入，加速 UI 开发。

可扩展架构

组件化页面与工具类封装，方便添加新题型或改进交互。
